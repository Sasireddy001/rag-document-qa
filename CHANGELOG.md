# Changelog

## [1.1.0] — 2026-07-22

### Added
- `ARCHITECTURE.md` with component diagram, data flow, scalability, fault tolerance, and deployment model.
- `SYSTEM_DESIGN.md` with requirements, functional design, NFRs, security, tradeoffs, and design decisions.
- Refined deployment notes for Docker Compose and cloud options.

## [1.0.0] — 2026-07-01

### Added
- Initial RAG Document QA chatbot with FastAPI and Streamlit interfaces.
- ChromaDB vector store with sentence-transformer embeddings.
- Support for OpenAI and Ollama LLM backends.
- Document ingestion for PDF, TXT, and Markdown files.
- `pytest` test suite with mocked LLM and embedding calls.
- GitHub Actions CI for linting and tests.
